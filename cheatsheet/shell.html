cheatsheet

Which directory am I in?

  • windows: cd
  • OS X: pwd

What files and directories are in this directory?

  • Windows:dir
  • OS X: ls
  • ls -l gives you detailed info

Changing directories:

    Action: to go into a directory
  • Windows cd dir-name
  • OS X: cd dir-name
    Action: To go up one directory
  • Windows: cd..
  • OS x: cd ..
