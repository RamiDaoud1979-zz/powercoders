What are the useful commands?

Windows OSX & linux Action
cd pwd Print working directory
dir ls Listing the directory
cd dir cd dir To go in to a directory
cd.. cd.. To go up one directory
mkdir dir mkdir dir Create directory
rmdir dir rmdir dir Delete a directory

Can I type any Character?

  • Safe
    • a-z 0-9 - _ .
  • OK
    • A-Z
  • Unsafe
    • Everything else
    • Especially ' " ( ) [ ] < > Space

Any shortcuts?

  • Cursor keys and Tab
    • ↑ and ↓ - Up and down through command history.
    • ← and → - Left and right through the command line.
    • Tab - completes on filenames and directories.
